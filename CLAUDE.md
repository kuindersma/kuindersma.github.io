# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Overview

This is a plain HTML/CSS personal portfolio site hosted on GitHub Pages (`kuindersma.github.io`). There is no build system, package manager, or framework — changes to `index.html` or `stylesheet.css` are deployed directly via git push to the `main` branch.

## Structure

- `index.html` — the entire site (single page)
- `stylesheet.css` — all styles
- `images/` — profile photo and favicon

## Development

Open `index.html` directly in a browser to preview changes. No build step required.

To deploy: push to `main` on GitHub — GitHub Pages serves the site automatically.
